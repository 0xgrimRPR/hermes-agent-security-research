# Disclaimer

## Scope Statement

This security research is conducted exclusively against a **self-hosted, local installation** of Hermes Agent (open-source, MIT License) in a controlled laboratory environment. No production systems, third-party infrastructure, live user data, or Nous Research infrastructure are targeted at any point during this assessment.

## Responsible Disclosure

Any vulnerabilities discovered during this research will be reported through responsible disclosure channels **before** public documentation:

- **Primary:** GitHub Security Advisories on the [Hermes Agent repository](https://github.com/NousResearch/hermes-agent/security)
- **Secondary:** security@nousresearch.com
- **Policy reference:** Hermes Agent's [Security Policy](https://github.com/NousResearch/hermes-agent/security/policy)

Public findings in this repository will only include information that has been:

1. Reported to Nous Research through the channels above
2. Given reasonable time for remediation (minimum 90 days)
3. Cleared for public disclosure, OR already publicly known (e.g., referenced in existing GitHub issues, CVEs, or published research)

## Legal

- All testing is performed against locally installed, open-source software under the MIT License
- No reverse engineering of proprietary components is involved
- No unauthorized access to external systems is attempted
- No production messaging platform terms of service are violated (test bots on isolated accounts only)
- This research falls within the scope of security research as defined by Hermes Agent's own [security policy](https://github.com/NousResearch/hermes-agent/security)
